1. Jelaskan apa yang dimaksud dengan NoSQL menggunakan pemahaman yang kalian ketahui!
    NoSQL adalah jenis database yang tidak memakai tabel dan baris seperti database tradisional.
    NoSQL dibuat untuk menyimpan data yang tidak selalu rapi, seperti dokumen, file log, atau data media sosial.

2. Jelaskan kapan harus menggunakan NoSQL dan Relational Database Management System!
    b. NoSQL:
        - Datanya tidak rapi, misalnya postingan media sosial.
        - Aplikasi butuh kecepatan tinggi dan bisa menangani banyak pengguna sekaligus.
        - Struktur data bisa berubah-ubah, misalnya formatnya belum pasti.
        (Contoh: Aplikasi chat, sistem rekomendasi seperti yang ada di e-commerce.)
    b. RDBMS:
        - Datanya terstruktur dengan rapi, seperti daftar nama, alamat, dan tanggal.
        - Perlu aturan ketat supaya data tidak berantakan (contoh: transaksi bank).
        - Hubungan antar data sangat penting (contoh: pelanggan dan pesanan).

3.Sebutkan contoh 2 tools/platform NoSQL selain ElasticSearch beserta Kelebihan tools/platform tersebut!
    a. MongoDB:
        - Kelebihan:
            - Fleksibel, cocok untuk data yang tidak selalu sama formatnya.
            - Mudah dipakai untuk aplikasi modern.
    b. Redis:
        - Kelebihan:
            - Kecepatan sangat tinggi, ideal untuk caching (penyimpanan sementara data).
            - Mendukung berbagai struktur data seperti string, list, dan hash.

4. Jelaskan apa yang Anda ketahui dari Airflow menggunakan pemahaman dan bahasa Anda sendiri!
    Apache Airflow adalah sebuah platform untuk menjadwalkan dan mengatur data workflow.
    Airflow digunakan untuk mengotomatisasi dan mengelola proses ETL (Extract, Transform, Load) atau pipeline data lainnya.

5. Jelaskan apa yang Anda ketahui dari Great Expectations menggunakan pemahaman dan bahasa Anda sendiri!
    Great Expectations (GE) adalah framework open-source untuk data validation dan data quality monitoring, 
    seperti melakukan pengecekan barang di pabrik, GE memeriksa data kita sesuai aturan yang sudah ditentukan.
